# Haze / VALUES public website

Static browser version of the Haze Seas companion.

## GitHub Pages publishing
1. Create a new **public** GitHub repository named `haze-values`.
2. Upload/push the contents of this folder to the repository root.
3. In the repository, open **Settings → Pages**.
4. Set **Source** to `Deploy from a branch`, choose `main`, and choose `/(root)`.
5. Save. GitHub will provide a live URL similar to `https://YOUR-USERNAME.github.io/haze-values/`.

After the site is live, submit that exact URL to Google Search Console for faster indexing.

The website is a fan-made companion; it does not claim affiliation with Roblox or Haze Seas.
